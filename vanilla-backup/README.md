# Ashutosh Ahirwal | Senior Drupal Frontend Developer Portfolio

A premium, interactive portfolio website built for **Ashutosh Ahirwal**, a Senior Drupal Frontend Developer with 5.1+ years of experience.

## ✨ Features
- **Premium Design**: Modern dark theme with a "Blueprint" aesthetic.
- **Interactive Hero**: Floating tech icons with mouse-parallax effect and a stunning abstract background.
- **Project Showcase**: "Browser Window" cards to highlight high-impact Drupal projects.
- **Performance**: Built with semantic HTML5, Vanilla CSS (HSL-based), and lightweight JavaScript.
- **Scroll Animations**: Smooth scroll and reveal effects using the Intersection Observer API.
- **Responsive**: Fully optimized for Desktop, Tablet, and Mobile.

## 🚀 How to Run Locally
1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Open your terminal in this directory.
3. Run the following command:
   ```bash
   npm run dev
   ```
   *This will start a live-server and automatically open the site in your browser.*

## 📂 Project Structure
- `index.html`: Main structure and content.
- `styles.css`: Core design system and layout.
- `script.js`: Interactivity and animations.
- `hero-bg.png`: Custom-generated hero background image.

## 🏆 Portfolio Sections
- **Hero**: Personalized summary and Drupal expertise.
- **About**: Career overview and community contributions.
- **Work Experience**: Timeline of professional history.
- **Projects**: Highlights of major Drupal builds (Zayed University, Apollo Hospitals, etc.).
- **Skills**: Visual indicators of technical mastery.
- **Blogs**: Direct links to published Drupal insights on Specbee.
